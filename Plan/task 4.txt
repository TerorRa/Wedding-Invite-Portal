Створи invite.php.

Сторінка має відкриватися за посиланням:

invite.php?code=XXXX

Логіка:
1. Отримати code з GET.
2. Знайти гостя в таблиці guests по invite_code.
3. Якщо гість не знайдений — показати красиве повідомлення "Запрошення не знайдено".
4. Якщо знайдений:
   - якщо status = invited, змінити status на opened
   - записати opened_at, якщо він ще порожній
   - додати запис у invite_logs з action = opened_invite

На сторінці показати:
- персональне звернення з іменем гостя
- дату весілля: 1 серпня 2026
- місце: Petrovskyi Brovar
- короткий текст запрошення
- таймлайн події
- блок дрес-коду
- форму RSVP

Форма має поля:
- will_attend: Так, буду / На жаль, не зможу
- plus_one: Буду сам / Буду з +1, якщо max_plus_one = 1
- plus_one_name, якщо вибрано +1
- drink: Вино / Шампанське / Віскі / Безалкогольне / Інше
- food_notes
- need_transfer
- song_request
- wish

Форма має відправлятися методом POST у submit_rsvp.php.
Разом із формою передавати invite_code.
